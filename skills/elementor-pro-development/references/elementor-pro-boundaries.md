# Elementor Pro Boundaries

Use this reference when deciding whether a task belongs in Elementor Pro, Elementor Free, Hello Elementor, or generic WordPress skills.

## Elementor Pro Owns

- Reusable addon or plugin code for dynamic tags, form actions, form fields, forms hooks, and Theme Builder condition APIs.

## Route To Elementor Free When

- The task is about widgets, controls, editor panels, Finder, context menu, managers, or addon bootstrap with no Pro dependency.

## Route To Hello Theme Or Child When

- The task edits Hello theme code directly, including theme-side location wrappers, theme location display logic, filter hooks, or child-theme overrides.

## Route Outside Elementor When

- The task is generic plugin logic, generic theme work, or WordPress Core behavior.
