# Elementor Free Boundaries

Use this reference when deciding whether a task belongs in Elementor Free, Elementor Pro, Hello Elementor, or generic WordPress skills.

## Elementor Free Owns

- Reusable addon or plugin code for widgets, controls, editor extensions, Finder, context menu, addon bootstrap, managers, and base assets.

## Route To Elementor Pro When

- The task depends on form actions, form fields, dynamic tags, forms hooks, or Theme Builder conditions.

## Route To Hello Theme Or Child When

- The task is theme-owned, Hello-specific, a site-specific child theme override, or requires editing `hello-theme-main` or `hello-theme-child-master`.

## Route Outside Elementor When

- The task is generic plugin logic, generic theme rendering, or WordPress Core behavior.
