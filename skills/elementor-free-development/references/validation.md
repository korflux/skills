# Elementor Free Validation

Use this reference to validate Elementor Free changes narrowly.

## Minimum Checks

- Confirm the widget, control, panel, Finder item, or context menu entry registers where expected.
- Confirm the affected editor UI appears where expected.
- Confirm preview and or frontend output works where relevant.
- Confirm assets load only on the surfaces that need them.

## Completion Signal

- Registration works.
- Output works.
- The narrowest relevant executable validation has been run when available.
