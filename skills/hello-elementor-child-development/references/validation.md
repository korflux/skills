# Hello Elementor Child Validation

Use this reference to validate Hello Elementor child theme changes narrowly.

## Minimum Checks

- Confirm the child override works.
- Confirm parent styles or scripts still load in the intended order.
- Confirm the change remains site-specific and child-theme scoped.

## Completion Signal

- The child theme behavior works.
- Parent compatibility remains intact.
- The change does not need to move into the parent theme or addon layer.
