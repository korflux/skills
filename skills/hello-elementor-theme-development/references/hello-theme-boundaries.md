# Hello Elementor Theme Boundaries

Use this reference when deciding whether a task belongs in the Hello parent theme, child theme, Elementor addon, or generic WordPress skills.

## Hello Parent Theme Owns

- Generic package-wide theme behavior in `hello-theme-main`, parent hooks, theme supports, settings, theme-side location registration or display, and theme-owned Elementor compatibility.

## Route To Hello Child Theme When

- The task is site-specific, local, or a light override that should not modify the parent package.

## Route To Elementor Addon Skills When

- The task is reusable addon behavior such as widgets, controls, dynamic tags, form actions, form fields, Pro integrations, or editor extensions.

## Route Outside Hello When

- The task is generic WordPress plugin logic, generic theme work unrelated to Hello, or WordPress Core behavior.
