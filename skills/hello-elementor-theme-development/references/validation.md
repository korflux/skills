# Hello Elementor Theme Validation

Use this reference to validate Hello Elementor parent theme changes narrowly.

## Minimum Checks

- Confirm the affected Hello hook or theme behavior still works.
- Confirm Elementor-dependent behavior still works when relevant.
- Confirm the change still makes sense as parent theme behavior rather than child-theme customization.

## Completion Signal

- Theme behavior works.
- Elementor compatibility still works where relevant.
- Parent theme ownership remains correct.
